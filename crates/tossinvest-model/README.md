# tossinvest-model

Pure, runtime-free data types for the [Toss Securities Open API](https://developers.tossinvest.com/docs).

Part of the [`tossinvest`](https://crates.io/crates/tossinvest) crate family. No I/O, no async,
no HTTP — just the serde models, domain newtypes, open enums, and order-lifecycle types.

> **Status: early scaffolding / name reservation.** See the
